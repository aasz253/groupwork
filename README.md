# Sifuna Codex Portfolio

This is a modern, responsive personal portfolio website for Sifuna Codex. It showcases skills, projects, testimonials, resume, and contact information. The site is built with HTML and CSS, featuring:

- Responsive navigation with a mobile sidebar menu
- Animated scrolling words and color transitions on the homepage
- Interactive icons linking to relevant resources
- Downloadable and viewable PDF resume
- Sample projects and testimonials
- Clean, modern UI with smooth animations

## Features
- **Responsive Design:** Works on phones, tablets, and laptops
- **Sidebar Navigation:** Hamburger menu for mobile
- **Animated Effects:** Scrolling words, icon animations, and color transitions
- **Project Showcases:** Example projects with descriptions
- **Contact Section:** Email, WhatsApp, LinkedIn, and more
- **Resume:** View or download as PDF

## Usage
1. Clone or download this repository.
2. Open `index.html` in your browser to view the portfolio.
3. Edit the HTML files to update your information, skills, and projects.
4. Replace the sample PDF with your own resume if needed.

## File Structure
- `index.html` - Homepage with animated intro
- `about.html` - About section
- `skilss.html` - Skills and sample projects
- `project.html` - Project portfolio
- `samples.html` - Code samples
- `testmonials.html` - Testimonials
- `resume.html` - Resume and achievements
- `contact.html` - Contact information
- `style.css` - Main stylesheet
- `Anthony_Sifuna_Web_Developer_Resume.pdf` - Resume PDF

## Customization
- Update icons, links, and text as needed
- Add your own projects and testimonials
- Change colors and animations in `style.css`
